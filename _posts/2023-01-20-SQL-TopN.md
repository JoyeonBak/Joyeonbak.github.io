```SQL Server
SELECT TOP 10 EMAIL, ID
FROM TBL; 
```


```MSSQL
SELECT EMAIL, ID
FROM TBL
WHERE ROWNUM<=N;
```

```MySQL
SELECT EMAIL, ID
FROM TBL
LIMIT N;
```
