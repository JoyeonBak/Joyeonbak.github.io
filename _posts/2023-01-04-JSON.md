## 파이썬 특징
JSON은 JavaScript Object Notation의 약자입니다.
JSON은 좀 더 쉽게 데이터를 교환하고 저장하기 위하여 만들어진 텍스트 기반의 데이터 교환 표준입니다.

```javascript
{
    "language": [
        {
            "name": "HTML",
            "category": "web",
            "developer": "W3C"
        },
        {
            "name": "Java",
            "category": "application",
            "developer": "Oracle"
        },
        {
            "name": "Python",
            "category": "application",
            "developer": "Python"
        }
    ]
}
```

  
* 간결하다
  규칙에 맞게 작성(들여쓰기)
* 프로그래밍을 즐기게 해준다
* 개발 속도가 빠르다
